About Riso Obscura

Riso-Obscura is a project that includes a Risograph photobooth program and a zine. It aims to introduce new Riso users to: operate a Risograph printer on a technical level and learn about spot color separation.

You can access the zine via a purchasing a physical copy, download the free .pdf or view it's online ressources.
